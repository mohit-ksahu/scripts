# scripts

Random scripts I build for myself, shared here in case they're useful to someone else too.

## Projects

- **[AIModels](./AIModels)** — Local LLM chat loop using `llama-cpp-python` to run GGUF models with streaming responses.
- **[AudioFX](./AudioFX)** — FFmpeg-based audio cleanup: silence removal, EQ, compression, and loudness normalization.
- **[Chatterbox](./Chatterbox)** — Voice cloning text-to-speech using Chatterbox Turbo, with automatic long-text chunking.
- **[CodeCarbon](./CodeCarbon)** — Zero-dependency, single-file CPU energy and CO₂ footprint tracker.
- **[Whisper](./Whisper)** — Speech-to-text transcription using OpenAI Whisper via Hugging Face Transformers.
